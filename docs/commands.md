#  Command Reference

X-UserBot comes with a variety of commands categorized for easy use.

##  Media Downloaders
| Command | Usage | Description |
|---------|-------|-------------|
| `/yt` | `/yt [link]` | Download YouTube videos or audio. |
| `/ig` | `/ig [link]` | Download Instagram Reels and Posts. |
| `/tt` | `/tt [link]` | Download TikTok videos (No watermark). |
| `/pin` | `/pin [link]` | Download Pinterest images and videos. |
| `/tw` | `/tw [link]` | Download Twitter (X) videos. |
| `/spotify` | `/spotify [query]` | Search and download Spotify tracks. |

##  Artificial Intelligence
| Command | Usage | Description |
|---------|-------|-------------|
| `/ai` | `/ai [question]` | Chat with Google Gemini AI. |
| `/ocr` | `/ocr [reply to image]` | Read text from an image. |
| `/genimage` | `/genimage [prompt]` | Generate images using AI. |

##  Group Management
| Command | Usage | Description |
|---------|-------|-------------|
| `/kick` | `/kick @user` | Remove a user from the group. |
| `/promote` | `/promote @user` | Make a user an admin. |
| `/setwelcome` | `/setwelcome [text]` | Set a custom welcome message. |
| `/mute` | `/mute [on/off]` | Stop the bot from responding in the group. |

##  System & Tools
| Command | Usage | Description |
|---------|-------|-------------|
| `/help` | `/help` | Show categorized list of commands. |
| `/alive` | `/alive` | Check if the bot is online. |
| `/purge` | `/purge [number]` | Delete multiple messages quickly. |
