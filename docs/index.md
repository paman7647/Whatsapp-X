# Welcome to WhatsApp-X 

**WhatsApp-X** is a professional-grade WhatsApp automation suite created by **Aman Kumar Pandey**. Designed for simplicity and power.

##  Key Features

- ** Universal Support**: Runs on Windows, Mac, Linux, and Android (Termux).
- ** One-Device Pairing**: Link your account using an 8-digit code (no QR scan needed!).
- ** Smart AI**: Integrated with Google Gemini for intelligent conversations.
- ** Media Suite**: Lightning fast downloads for YouTube, TikTok, Instagram, and more.
- ** Database Power**: Built-in support for local and cloud MongoDB.
- ** Cloud Ready**: Fully optimized for Render, Heroku, and Docker.
